# SmileSchool Project
This repository contains the **HTML & CSS Advanced Assignment** for ALX HTML/CSS course.
## Structure
- `css_advanced/` → Folder containing the main HTML and CSS files
  - `index.html` → Main HTML file
  - `styles.css` → Stylesheet for the project
## Description
This is a mock e-learning platform called **SmileSchool**. It features:
- Header and navigation
- Banner with pros section
- Quote section
- Video tutorials
- Membership plans
- FAQ
- Footer with social links
All styles are in `styles.css`.
